## Our Standards

Every app we ship meets the same baseline:

- **Zero ads.** No advertising SDK. No banner, interstitial, or rewarded ad — ever.
- **Zero telemetry.** No analytics. No crash reporting to external servers. Nothing leaves your device.
- **Offline-first.** Core functionality works without an internet connection.
- **Open source.** GPL-3.0. Every line of code is public and auditable.
- **No account required.** You do not need to register to use any of our apps.

## Tech Stack

- **Mobile:** Kotlin (Jetpack Compose), Java, Flutter
- **Architecture:** Clean Architecture, MVVM, MVI
- **Principles:** Privacy-by-design, local-first data storage, minimal permissions
